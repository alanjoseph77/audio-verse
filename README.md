# SonicShelf üéß

A modern, beautiful audiobook player built with **React Native**, **Expo**, and **Reanimated**. 
This app aggregates public domain audiobooks from **LibriVox** (via the Internet Archive), providing a premium listening experience for free content.

## ‚ú® Features

- **Dynamic Library**: Browse curated collections (Romance, Classic, Sci-Fi) and search the vast LibriVox archive.
- **Modern UI**: Smooth animations, glassmorphism effects, and custom 3D-style loaders.
- **Audio Player**: Full-featured player with queue management, background playback, and lock screen controls.
- **Queuing System**: Seamlessly play through chapters with auto-advance functionality.
- **Artist Profiles**: Explore books by your favorite authors.

## üì± Tech Stack

- **Framework**: [Expo](https://expo.dev) & React Native
- **Navigation**: [Expo Router](https://docs.expo.dev/router/introduction)
- **Animations**: [React Native Reanimated](https://docs.swmansion.com/react-native-reanimated/)
- **Audio**: [Expo AV](https://docs.expo.dev/versions/latest/sdk/av/)
- **Icons**: Ionicons

## üöÄ Get Started

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Start the app**
   ```bash
   npx expo start
   ```

3. **Run on Device**
   - Scan the QR code with **Expo Go** (Android/iOS).
   - Or press `a` for Android Emulator / `i` for iOS Simulator.
